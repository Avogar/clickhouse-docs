# Integrations

:::note
[WIP] We are currently actively compiling the list of ClickHouse integrations below so it's not yet exhaustive. Feel free to [contribute](https://github.com/ClickHouse/clickhouse-docs#how-to-contribute) any relevant ClickHouse integration to the list.

In the meantime, the interfaces list is still available [here](https://clickhouse.com/docs/en/interfaces).
:::

ClickHouse integrations are organized into **Language client**, **Data ingestion**, **Data visualization**, **SQL client**, and **Migration** categories.  Additoinally, the integrations are organized by their support category: **Core**, **Partner**, and **Community**.

<span style={{fontSize: '1.5rem', fontWeight: 'bold'}}>Core integrations</span>

Core integrations are built or maintained by ClickHouse. These integrations are supported by ClickHouse and live in the ClickHouse GitHub organization.

<div class="adopters-table">

|Name|Logo|Category|Description|Resources|
|------|----|----------------|------------------|-------------|
|Amazon S3|<S3SVG style={{width: '3rem', height: '3rem'}} />|Data ingestion|Import from, export to, and transform S3 data in flight with ClickHouse built in S3 functions and clickhouse-local.|[Documentation](./s3)|
|ClickHouse Client|<ClickHouseSVG style={{width: '3rem'}} />|SQL client|ClickHouse Client is the native command-line client for ClickHouse.|[Documentation](../interfaces/cli)|
|DBeaver|<DbeaverSVG style={{width: '3rem'}} />|SQL client|Free multi-platform database administration tool. Connects to Clickhouse through JDBC driver.|[Documentation](./sql-clients/dbeaver)|
|dbt|<DbtSVG style={{width: '3rem'}} />|Data ingestion|Use dbt (data build tool) to transform data in ClickHouse by simply writing select statements. dbt puts the T in ELT.|[Documentation](./dbt)|
|Go|<GoLangSVG style={{width: '3rem'}} />|Language client|The Go client uses the native interface for a performant, low-overhead means of connecting to ClickHouse.|[Documentation](https://github.com/clickhouse/clickHouse-go#readme)|
|Java|<JavaSVG style={{width: '3rem', 'background-color': 'white'}} />|Language client|The Java client is an async, lightweight, and low-overhead library for ClickHouse.|[Documentation](https://github.com/clickhouse/clickhouse-jdbc#readme)|
|Kafka|<KafkaSVG style={{width: '3rem'}} />|Data ingestion|Bidirectional integration with Apache Kafka, the open-source distributed event streaming platform.|[Documentation](./kafka)|
|Python|<img src={require('./images/logos/python-logo.png').default} class="image" alt="Python logo" style={{width: '3rem', 'background-color': 'transparent'}}/>|Language client|A suite of Python packages for connecting Python to ClickHouse.|[Documentation](https://github.com/clickhouse/clickhouse-connect#readme)|
|Superset|<SupersetSVG style={{width: '3rem', 'background-color': 'white'}} />|Visualization client|Explore and visualize your ClickHouse data with Preset or Apache Superset.|[Documentation](../connect-a-ui/superset-and-clickhouse)|

</div>
